# JPMorgan Financial Statement Analysis

This project demonstrates financial statement literacy through analysis of JPMorgan Chase & Co.'s financial statements and 10-K disclosures.

## Project Components

Financial Analysis Memo  
A two-page analyst memo summarizing insights from JPMorgan's financial statements.

Ratio Dashboard  
An Excel dashboard presenting key financial ratios and revenue trends.

## Financial Metrics Covered

Return on Equity (ROE)  
Return on Assets (ROA)  
Efficiency Ratio  
Revenue Growth  

## Skills Demonstrated

Financial Statement Analysis  
10-K Interpretation  
Ratio Analysis  
Financial Reporting  
Business Writing
